# ３目並べ


1. Googleで検索(「stack haskell 環境構築」とか)してStackをinstall
3. git cloneしてくる
4. クローンしたディレクトリに移動してターミナルで`stack build`入力
6. ビルドに成功したらプロジェクトのルートディレクトリで`stack runhaskell app/Main.hs`を入力してプレイ可能

ターミナル上では0~8の数字を入力することで駒を設置できる
```
    |       |
 0  |    1  |   2
------------------
    |       |
3   |   4   |   5
------------------
    |       |
6   |   7   |   8
```
