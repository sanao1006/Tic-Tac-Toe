# ３目並べ
を作った

1. Googleで検索してStackをinstall
2. ターミナルで`stack install`入力
3. git cloneしてくる
4. クローンしてきたこのディレクトリ内で`stack build`入力
5. ビルドに成功したらプロジェクトのルートディレクトリで`stack runhaskell app/Main.hs`を入力してプレイ可能

ターミナル上では0~8の数字を入力することで駒を設置できる
```
    |       |
 0  |    1  |   2
------------------
    |       |
3   |   4   |   5
------------------
    |       |
6   |   7   |   8
```
